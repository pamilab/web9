2. Design and Simulation

FMCW Chirp Signal

Linear frequency-modulated continuous wave used to encode target depth information into beat frequency.

Equation Expression Description
STFT S(f, t) = ∑τ=0N−1 x(τ) · w(t − τ) · e−j2πfτ Short-Time Fourier Transform used to extract time–frequency information from the beat signal.
Beat Frequency fb = f2 − f1 Difference between transmitted and received frequencies after mixing.
Range Estimation R = (c · T · fb) / (2B) Depth calculation using beat frequency, where c is propagation velocity, T is chirp duration, and B is bandwidth.
Range Resolution ΔR = c / (2B) Minimum resolvable depth separation determined by FMCW bandwidth.

3. Hardware Integration

RF Front-End

In the initial phase, the RF components were carefully interconnected to establish the complete signal path of the radar system. Every high-frequency connection uses standard SMA connectors using 50-ohm impedance-matched devices.

SDR Digitization

The ADC on this SDR is used to digitize the beat frequency only and other functions will not be used. The sampling rate will be maximized and the Nyquist limit will be reached at maximum modulating frequency since it is using sampling. The USB will carry 1.28Gbps at 20MSPS sampling rate using complex64 data type which contains 32 bits each of I and Q samples. This was chosen because it is the cheapest available digitizer with very high sampling rate, increasing the bandwidth limits, hence range resolution.

Hamming Window

The Hamming window is a mathematical function used in FMCW-GPR System. It is applied to the beat signal before performing the Fourier Transform to reduce spectral leakage, which can distort the frequency analysis of the signal.
