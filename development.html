Building the Prototype

From concept and simulation to hardware integration and validation.

1. Concept and Planning

The project began with identifying system limitations in existing GPR designs and selecting FMCW radar as the optimal solution. System requirements, operating frequency, and compliance constraints were defined.

2. Design and Simulation

Radar equations, chirp characteristics, and depth-to-frequency relationships were analyzed to predict system performance before hardware construction.

3. Hardware Integration

RF modules, control units, and digitization hardware were assembled into a unified radar front-end. Mechanical mounting ensured repeatable testing.

4. Software Development

Signal acquisition and processing were implemented using GNU Radio and Python, enabling real-time visualization and automated data handling.

5. Testing & Validation

The system was evaluated using controlled experiments with metallic targets embedded at known depths under a wooden platform.

Test Setup Parameters

  • Operating Frequency: 2.4 GHz ISM
  • Target Depths: 29 cm & 72 cm
  • Trials: Multiple per depth
  • Reference: Manual tape measure
